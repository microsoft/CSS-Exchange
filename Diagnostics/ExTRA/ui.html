The trace can be created, deleted, started, and stopped from the command line or PowerShell.

To delete a previously existing data collector set:

logman delete ExchangeDebugTraces

To create a data collector which is non-circular and stops at 1 GB:

logman create trace ExchangeDebugTraces -p "{79bb49e6-2a2c-46e4-9167-fa122525d540}" -o c:\tracing\trace.etl -ow -f bin -max 1024 -mode globalsequence

To create a data collector which is circular and stops at 2 GB:

logman create trace ExchangeDebugTraces -p "{79bb49e6-2a2c-46e4-9167-fa122525d540}" -o c:\tracing\trace.etl -ow -f bincirc -max 2048 -mode globalsequence

To create a data collector which is non-circular and creates a new file every 512 MB until you stop it manually:

logman create trace ExchangeDebugTraces -p "{79bb49e6-2a2c-46e4-9167-fa122525d540}" -o c:\tracing\trace.etl -ow -f bin -max 512 -cnf 0 -mode globalsequence

To start the trace:

logman start ExchangeDebugTraces

To stop the trace:

logman stop ExchangeDebugTraces

The collector can also be started and stopped from Perfmon.
